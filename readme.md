# Chat Arena - Remote

## Setup

Download [Node.js](https://nodejs.org/en/download/).

复制一份 `.env.example`,  修改为：`.env`, complete it，/很重要！！！

运行命令:

``` bash
# Install dependencies (only the first time)
npm install

# Run the local server in development mode
npm run dev

# Run the local server in live mode
npm run live
```
